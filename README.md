# My-Portfolio

A collection of small interactive web projects built with HTML, CSS, and JavaScript. Includes a Calculator, Rock-Paper-Scissors game, Coin Flip simulator, To-Do List, and a YouTube clone. Designed as practice projects and a showcase of front-end development skills.

---

## Projects Included

1. **Calculator** – A simple calculator with basic arithmetic operations and keyboard support.
2. **Rock-Paper-Scissors** – Play against the computer with live score tracking.
3. **Flip a Coin** – Simulate coin flips with visual feedback.
4. **To-Do List** – Add, delete, and mark tasks as completed.
5. **YouTube Clone** – Visual clone of YouTube’s homepage layout (static, no video playback).

---

## Features

- Responsive design  
- Keyboard and mouse interactions  
- Local storage support for saving scores and tasks  
- Clean, modern UI for all projects  

---

## How to View

1. Download or clone this repository.  
2. Open the `index.html` file in your web browser.
